# 支持的硬件

## Zigbee 适配器（adapter）

Zigbee2MQTT 支持多种适配器（adapter），用于与 Zigbee 无线网络进行通信。
大多数适配器通过 USB 端口连接，但也可以使用 GPIO 引脚，或通过 WIFI、以太网络 [远程连接](../advanced/remote-adapter/connect_to_a_remote_adapter.md)
使用 TCP 进行通信。参见 [支持的 Zigbee 适配器列表](./adapters/)

## Zigbee 设备

目前已有大量 [支持的 Zigbee 设备](../supported-devices/)，且数量仍在持续增长。
如果你的设备尚未被支持，可以查看 [如何支持新设备](../advanced/support-new-devices/01_support_new_devices.md)，一起为 Zigbee2MQTT 做出贡献。

# Home Assistant addon

::: warning 注意
Home Assistant 仅支持 64 位架构（aarch64 和 amd64）。这意味着诸如 Sonoff iHost（32 位）或老款 Raspberry Pi（armv7）之类的硬件无法使用。
:::

::: warning 注意
如果你使用的是 Raspberry Pi，请确保至少使用 Raspberry Pi 4，因为在 Raspberry Pi 3 上运行可能会因资源有限而导致不稳定。
:::

如果你运行的是 Home Assistant OS 或 Supervised 版本的 Home Assistant 实例，安装 Zigbee2MQTT 最简单的方式就是通过 addon。安装说明可以在[这里](https://github.com/zigbee2mqtt/hassio-zigbee2mqtt#installation)找到。
